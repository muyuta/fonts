# SCL Basic Latin

SCL Basic Latin is a simple and functional typeface that includes the Basic Latin character set, Turkish characters, 
Its minimal design makes it suitable for indie projects, prototypes, and personal designs, offering both readability and lightweight file size.

## Features
- Basic Latin character set
- Turkish character support (ç, ğ, ı, İ, ö, ş, ü)
